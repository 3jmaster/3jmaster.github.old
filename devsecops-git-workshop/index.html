What if your DNS is poisoned?
What if you typed wrong?
What if the script is nested?

Git is a software afterall, how you use it...

  • Do not used shared credentials, especially root
  • Enforce signing of commits
  • Do not check in secrets into source code - NO NOT EVEN DEVELOPMENT
  • Train developers to use variables and substitutions
  • Use .gitignore to prevent accidents
  • Keep your code infrastructure secure (and away from public whenever possible)

  • Security Best Practices
